RasEnumDevices

RasEnumDevices is a Visual Basic 6.0 application that displays all the RAS capable 
devices that have been configured for Dial Up Networking.

FILES
=====

The RasEnumDevices application uses the following files

File                  Description

README.TXT            Readme file for RasEnumDevices application
RASENUMDEVICES.VBP    VB6.0 project file
RASENUMDEVICES.FRM    RasEnumDevices form file
RASENUMDEVICES.BAS    Module that contains the function and structure declarations


RAS API's USED
==============

RASENUMDEVICES


----------------------------------------------------
RUNNING THE APPLICATION ON WINDOWS NT AND WINDOWS 9x
----------------------------------------------------

To run the application load the project in VB 6.0 and run it.

For Windows 95(pre OSR2) you need to install DUN 1.2x upgrade
