Microsoft Agent

Java Samples - Hello World 2

This sample implements a notify sink so that the applet can get event notifications from the Microsoft Agent server. If a default character is successfully loaded, the sample adds a command so that the user can select a different character via the character popup menu.

Try left clicking on the character to see what happens.

If you have default characters installed on your system (Genie, Merlin, Robby, or Peedy) in the %window%\msagent\chars directory, right click on the character and select Choose Character.
