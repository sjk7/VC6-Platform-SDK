Fundamentals

Overview

The Fundamentals directory contains a wide variety of samples demonstrating how to use COM to build applications. The samples in this directory are geared towards COM programmers who are using COM Base features. "COM Base" refers to the COM features preceding the introduction of COM+ services. For more information on some of the COM Base samples, see the following topics:

  •  Non-Blocking Method Calls (Asynchronous COM)
  •  Call Cancellation
  •  Custom Surrogate
  •  Lightweight Proxy Handler
  •  Method Call Timeouts
  •  DCOM Pipes
  •  COM Tutorial Samples
