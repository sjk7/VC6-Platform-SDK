MTX Stop

Overview

MTXStop.exe shut's down all applications on a machine including the system app (last of course).

To Build & Run:

- Open MTXStop.Vbp and build MTXStop.exe
- View components running in the Component Services Explorer
- Run MTXStop.exe and watch all the Components stop
