Create Empty COM+ Applications

This procedure creates one empty COM+ application, OPBank.

  1. Start the Component Services Explorer. Click Start, point to Programs, point to Administrative Tools, and then click Component Services.
  2. Select Console Root, click Component Services, click Computers, click My Computer, and then click COM+ Applications.
  3. Right-click the COM+ Applications folder, select New, and then click Application.
  4. When the wizard appears, click Next.
  5. Click Create an empty application.
  6. When you see the Enter a name for the new application prompt, type OPBankM into the edit box. Make sure that the Server application option button is clicked and then click Next.
  7. Make sure that the Interactive user - the current logged on user option button is clicked and click Next.
  8. Click Finish.
