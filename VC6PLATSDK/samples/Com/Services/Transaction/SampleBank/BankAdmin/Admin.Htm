Admin Scripts

Overview

The admin scripts demonstrate how to use the Admin scripting components to create COM+ applications and install components.  All of the functionality in the COM+ Administration snap-in is available from the scripting components.

Files

Description

ExportApplication.vbs

Scripts that exports the “Sample Bank” and its components so it can be imported on another server.  Creates .msi and .cab files.

ImportApplication.vbs

Imports a previously exported Appplicayion.  Reads .msi and .cab files.

InstallSampleBank.vbs

Creates the “Sample Bank” application and installs the components that it is comprised of.

UninstallSampleBank.vbs

Deletes the “Sample Bank” application and its components.
