Set the Queued property for the QCShip and QCNotify applications

This sequence creates the underlying Message Queuing queues that provide the queuing services for these queued components. You can see the effect of setting the Queued property on the COM+ application by running the Message Queuing MMC snap-in (see Important Message Queuing Information) and looking in the Public Queues folder.

  1. Start the Component Services Explorer. Click Start, point to Programs, point to Administrative Tools, and then click Component Services.
  2. Select Console Root, click Component Services, click Computers, click My Computer, and then click COM+ Applications.
  3. Right-click QCShip and then click Properties.
  4. Click the Queuing tab.
  5. Click Queued.
  6. Do not select Listen.
  7. Click OK.
  8. Repeat the procedure for QCNotify.
