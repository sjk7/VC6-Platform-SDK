For each COM+ application install the appropriate component

In this procedure we will add the following components to their respectively named applications:

    • QCOrder
    • QCShip
    • QCNotify
  1. Start Component Services Explorer by clicking Start, point to Programs, point to Administrative Tools and then click Component Services.
  2. Select Console Root, click Component Services, click Computers, click My Computer, click COM+ Applications, click QCOrder, and then click Components.
  3. Right-click the Components folder, click New and then click Component.
  4. When the wizard appears, click Next.
  5. Click Install new component.
  6. Use the dialog box to navigate to [Platform SDK Root]\samples\com\services\Queued_Components\QCSamp1\QCOrder\Debug and click QCOrder.dll.
  7. Click Next.
  8. Click Finish.
  9. Repeat the procedure for QCShip and QCNotify.
