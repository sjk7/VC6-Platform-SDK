OVERVIEW

The User Events sample shows how to generate application specific events. The events can be monitored using the ComSpy tool located in the Administration directory. The sample is based on the extensive SampleBank sample from the Transaction directory..

The User Events sample is made up of four programs:

VBBank , a client application that creates an instance of a Bank.Account server and calls the Post method.

vbacct , a Visual Basic version of the Bank.Account COM+ server.

vcacct , a Visual C++ version of the Bank.Account COM+ server.

BUILDING THE SAMPLE

Build each of the three Bank.Account servers.

  • Visual C++:  Open the workspace account.dsw in the vcacct folder and build with F7.
  • Visual Basic: Build vbacct.vbp in the vbacct folder.

Each folder will then have on DLL: vcacct.dll & vbacct.dll .

Run the included script InstallSampleBank.vbs to create a COM+ application and install the necessary servers.

OPERATION

Build and launch the ComSpy.exe  tool in the \Administration\Spy\ComSpy folder.

Start Bank Client by running vbbank.exe in the vbbank folder.

In ComSpy, select the Sample Bank application using Spy | Select Events... menu option. Select to monitor UserDefined events and press the Close button.

By pressing the Post button Bank Client, events can be observed in ComSpy.
